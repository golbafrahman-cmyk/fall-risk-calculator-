راهنما برای تهیه نسخه اندروید (PWA یا APK)
-----------------------------------------

1) نصب سریع به‌عنوان PWA (بدون ساخت APK)
- فایل‌های داخل این پوشه را روی یک وب‌سرور یا GitHub Pages آپلود کنید.
- با مرورگر کروم روی اندروید آدرس سایت را باز کنید.
- از منوی مرورگر گزینه 'Add to Home screen' یا 'Install app' را انتخاب کنید.
  اپ به‌صورت standalone نصب خواهد شد.

2) تولید APK/AAB با PWABuilder
- به https://www.pwabuilder.com بروید.
- URL سایت خود را وارد کنید (یا ابتدا فایل‌ها را هاست کنید).
- مراحل تولید Android APK/AAB را دنبال کنید و فایل را دانلود کنید.

3) ساخت با Android Studio (برای توسعه‌دهندگان)
- پروژه جدید ایجاد کنید و یک Activity با WebView بسازید که index.html را بارگذاری کند.
- بیلد و خروجی APK بگیرید.
